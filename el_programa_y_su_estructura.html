El programa y su estructura

ESTRUCTURA BÁSICA DE UN PROGRAMA EN PYTHON:

La estructura básica de un programa hecho en Python es tan simple que se asemeja a un seudocódigo haciéndolo tan simple y fácil de aprender que lo hace el lenguaje ideal para aprender a programar. C/C++ que es el lenguaje base de donde proviene Python, aunque algunos dicen que es la simplificación de Java, pero a la final este proviene del anterior y así sucesivamente, por lo tanto C/C++ debería ser un poco más compleja su estructura básica y es así:

#include<iostream>
using namespace std;
int main(){
      cout<<”Este es un mensaje de salida en pantalla”;
      }

Ya tenemos la estructura básica teniendo en cuenta que el código fuente realmente comienza y lo colaremos entre esas dos llaves, pero veremos que la primera línea es la inclusión de la librería básica de C++ (iostream) donde se invoca las funciones de entrada y salida por terminal (cin y cout), pero en Python esto es mucho más simple:

     print(“Este es un mensaje de salida en pantalla”)

Seudocódigo:

Algoritmo ProgramaInicial
Escribir "Este es un mensaje de salida en pantalla"
FinAlgoritmo

C:

#include<stdio.h>
int main() {
     printf("Este es un mensaje de salida en pantalla\n");
     return 0;
    }

C#:

using System;
namespace prueba {
     class estructurabasica {
          static void Main(string[] args) {
          Console.WriteLine("Este es un mensaje de salida en pantalla");
          }
     }
}
