Lenguajes de Programación

Un programa es un algoritmo, pero la única característica de este que es capaz de diferenciarlos es que todo programa es un algoritmo pero no todo algoritmo es un programa ya que el algoritmo puede estar escrito en seudocódigo, mientras que el programa presenta una estructura más rigurosa a nivel de sintaxis y semántica ya que él mismo se escribe utilizando un lenguaje de programación de alto nivel para codificarlo.
