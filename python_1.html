Python 1

PYTHON

Python es un lenguaje de programación interpretado, llamado también de tipo script. Básicamente un lenguaje interpretado es aquel que puede ser ejecutado y traducido a través de un intérprete a diferencia de los lenguajes compilados que es traducido por un compilador y ejecutado por el procesador. Python fue creado alrededor de los años 90’s por Guido Van Rossum, hoy en día este científico de la computación trabaja en el Proyecto Dropbox.
Python se caracteriza por tener una sintaxis limpia gracias al uso de un código legible, evitando el uso del punto y coma y usando la identación en vez de las llaves para definir estructuras de control.
Es un lenguaje multiparadigma aunque creado para su uso en la Programación Orientada a Objeto también utiliza la Imperativa y la Funcional. Tiene un Tipado Dinámico esto se refiere a que su tipo de dato se define en tiempo de ejecución, al asignarse un valor el tipo de dato es definido por el tipo de dato reconocido en el valor asignado, es decir, si a una variable la definimos con un valor de 45 ella se asume como de tipo entero.
