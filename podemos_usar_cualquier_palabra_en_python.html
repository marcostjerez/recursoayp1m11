¿Podemos usar cualquier palabra en Python?

PALABRAS RESERVADAS EN PYTHON

Las palabras reservadas son aquellas que el programador no puede utilizar para nombrar variables, funciones, clases, colecciones de datos o cualquier otra estructura que deba ser nombrada por él, estas palabras son las sentencias, funciones, operadores o librerías, ejemplo: if, print, input, else, elif, entre otras que estaremos viendo en el transcurso del estudio del lenguaje.
