¿Cómo identificamos las variables?

IDENTIFICADORES DE VARIABLES:

Todas las variables que vamos a utilizar deben ser declaradas para que el intérprete sepa sus atributos y al momento de ejecutar no halla conflictos, para el identificador no podemos utilizar palabras reservadas del lenguaje que son palabras que ya están guardadas para su uso exclusivo como sentencias, operador, conector u algún otro símbolo dentro de Python. Se podrá utilizar para el identificador, letras, números y el símbolo del guion bajo, siempre empezando con letras (mayúscula o minúscula) pero por convenio utilizaremos las minúsculas para el nombre de las variables, se diferencian las mayúsculas y las minúsculas, es decir que la variable abc no es la misma ABC, se permite el uso de la ñ o la tilde para definir el nombre de una variable a partir de Python 3, pero por mantener las buenas prácticas de un programador tratemos de no usarlas.
